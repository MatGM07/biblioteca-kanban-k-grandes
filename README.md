# Sistema de Gestión de Biblioteca Universitaria
## Descripción del Proyecto

El Sistema de Gestión de Biblioteca Universitaria es una aplicación diseñada para facilitar la administración y control de los recursos bibliográficos de una institución académica. Este sistema permitirá gestionar libros, usuarios, préstamos, devoluciones y multas, optimizando los procesos internos de la biblioteca y mejorando la experiencia de los usuarios.

Este repositorio corresponde a la fase de planeación inicial del proyecto, cuyo objetivo principal es definir, organizar y gestionar las tareas mediante el uso de la metodología Kanban, empleando Issues como mecanismo de seguimiento.


##Objetivos

###Objetivo General

Planificar el desarrollo de un Sistema de Gestión de Biblioteca Universitaria utilizando la metodología Kanban para la organización y seguimiento de tareas.

###Objetivos Específicos

Definir los requerimientos funcionales principales del sistema.

Crear Issues que representen funcionalidades, tareas técnicas y actividades de soporte.

Organizar las tareas en un tablero Kanban.

Simular el flujo de trabajo de un equipo de desarrollo.

Familiarizarse con el uso de herramientas de gestión ágil.
